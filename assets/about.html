About this project

This project is inspired by Noah Veltman’s SF Streets map, with some additional features and layers on top. The latter comes from my love for my home town, one of the greatest cities in the world.

Data sources

The roads shapefile comes from Mapzen’s excellent metro extract service built on top of Open Street Map. Further data conversion was done using the ogr2ogr tool. The map itself is built with Leaflet JS, with some more help from a custom built of Mike Bostock’s D3.js library. Check out the code and data on Github.

Classification notes

The categorisation of the names are strictly subjective using what I thought would be apt categories. Any omission, mistake, or any other thing that you may find offensive is strictly coincidental. No intentional harm was intended.

  • The genders have been manually coded. I made a spreadsheet of every street name in the map shape and went line by line. There may be some cases where genders could not be assigned or I couldn’t verify the gender. Those have been coded to “no gender”.
  • Information about street names was also manually collected. This means I literally typed each name in Google and DuckDuckGo, and used whatever these two engines gave me. I have tried my level best to gather as much information as I could. I understand this is biased towards sources in English but going through 4,000-odd names is not an easy task.
  • The religion category has been applied to any name connected to any religion — a saint, missionary, or a spiritual leader
  • Politics was much more tricky. So I applied this logic. Any name that has relevance in context of events before Independence is historical. Anything after Independence is politics. So, Mahatma Gandhi would be historical. But Lal Bahadur Shastri would be political.

If you have any concerns or qualms with it, or you want to add information about any street, feel free to email me.
